# US-008 — Delete a Placed Block

**As a** user,
**I want to** remove a block from the timeline,
**So that** I can correct mistakes or clear time I've freed up.

## Acceptance Criteria

- [ ] Each placed block has a × delete button
- [ ] On desktop: the delete button appears on hover
- [ ] On mobile: the delete button is always visible (no hover state on touch)
- [ ] Clicking/tapping the × removes the block immediately
- [ ] The timeline empty state reappears if all blocks are deleted
- [ ] Deletion is saved to `localStorage` and the wallpaper re-renders
