# US-005 — Choose a Wallpaper Theme

**As a** user,
**I want to** choose a colour theme for my wallpaper,
**So that** it matches my phone's aesthetic or my mood.

## Acceptance Criteria

- [ ] Four themes are available: Dark, Navy, Purple, Paper
- [ ] Theme is selected by clicking a coloured dot in the Preview panel header
- [ ] The active theme dot shows a highlighted border
- [ ] The wallpaper canvas updates immediately when a theme is selected
- [ ] The selected theme is saved to `localStorage` and restored on next visit

## Themes

| Name | Background | Accent |
|---|---|---|
| Dark | Near-black (`#0e0e0e`) | Yellow-green (`#c8f072`) |
| Navy | Deep navy (`#0a1628`) | Sky blue (`#7eb8f7`) |
| Purple | Deep purple (`#1a0a2e`) | Lavender (`#c8a0f8`) |
| Paper | Off-white (`#f0ebe0`) | Forest green (`#2a5c2a`) |
